HAVE TO MAKE THEM STOP SO THEY CAN START AGAIN

By Kerri A. Fallat

Thoughts crammed against my skull.
Have to get them out.
Have to release them.
They are dragged and dropped into my brain
like files uploaded to a google drive.
Have to get them out.
Must get them out.
Have to escape the repetitive record that they spin on.
Have to write them down.
Have to list them.
I jot them down so quickly that I can barely recognize the scribbles on
The blank page
Or that glowing text that lingers in Visual Studio Code,
Bridged and nested
Between the cushionary body and html tags.
Have to get them out.
Have to write them all down.
So the thinking stops
So the unraveling haults
So the web can stop spinning
Just so they can start again
Tomorrow.
